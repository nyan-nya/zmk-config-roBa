# roBa ファームウェアカスタマイズ適用手順

今回のキーマップ変更（スクロール用キーのPキーへの移行）およびExcel横スクロールの有効化を行う手順です。

## 1. ファームウェアのビルドと適用

### Step 1: 変更内容のGitHubへのプッシュ
今回の変更を作業ディレクトリ（`C:\Users\yamam\.gemini\antigravity\scratch\zmk-config-roBa`）からご自身のGitHubリポジトリ（`nyan-nya/zmk-config-roBa`）へプッシュします。
※Gitコマンドが使用可能であれば、新しいブランチにコミットしてプッシュすることを推奨します。

### Step 2: GitHub Actionsでのビルド確認
1. GitHub上のご自身のリポジトリページを開きます：
   [nyan-nya/zmk-config-roBa](https://github.com/nyan-nya/zmk-config-roBa)
2. 画面上部の **「Actions」** タブをクリックします。
3. プッシュしたコミットに対するビルドワークフロー（例: `Build`）が自動的に開始されます。緑色のチェックマーク（成功）に変わるまでお待ちください（通常1〜3分程度）。

### Step 3: ファームウェア（.uf2）のダウンロード
1. 成功したビルドジョブをクリックして詳細ページに入ります。
2. ページ下部の **「Artifacts」** セクションに、ビルドされたファイル（例: `firmware`）が表示されます。
3. これをクリックしてZIPファイルをダウンロードし、展開します。

### Step 4: キーボードへの書き込み（フラッシュ）
展開すると、以下のファイルが含まれています：
* **`roBa_L-seeeduino_xiao_ble-zmk.uf2`** (左手用コントローラー用)
* **`roBa_R-seeeduino_xiao_ble-zmk.uf2`** (右手用コントローラー用)
* **`settings_reset-seeeduino_xiao_ble-zmk.uf2`** (接続リセット用 - 通常は使いません)

書き込み手順：
1. キーボードの **左手側** をUSBケーブルでPCに接続します。
2. 左手側のリセットスイッチ（またはボード上のリセットボタン）を **ダブルクリック**（素早く2回押す）します。
3. PCに **「XIAO-SENSE」** や **「NICENANO」** といった名前の外部ドライブとして認識されます。
4. **`roBa_L-seeeduino_xiao_ble-zmk.uf2`** をそのドライブにドラッグ＆ドロップ（またはコピー＆ペースト）します。書き込みが終わると自動的にドライブが切断されます。
5. 同様に、USBケーブルを **右手側** に差し替え、ダブルクリックしてドライブを開き、**`roBa_R-seeeduino_xiao_ble-zmk.uf2`** を書き込みます。

---

## 2. Excelでの横スクロールの有効化

1. 今回作成した **`excel_scroll_helper.ahk`** をダブルクリックして実行します。
   * ※あらかじめPCに **AutoHotkey** (v1.1またはv2) がインストールされている必要があります。
2. 実行すると、Excel上でトラックボールを左右に動かした際、自動的にExcel側で横スクロール（Shift＋縦スクロール相当）に変換され、スムーズに左右へ移動できるようになります。
3. 常に有効にしたい場合は、このファイルをWindowsのスタートアップフォルダ（`Win+R` で `shell:startup` を実行して開くフォルダ）に配置してください。
